खाजोना होग्रानि थि खालामनायनि सरासनस्रा थावनिखौ गनायनाय जायो। दालाय दफा (1) जों बेखौ बिथोन होनाय जायो दि जेराव सासे खाजोना लाग्राया जायखिजाया जायगायाव फालांगि, जिउ राहा एबा जिउ राहाखौ सामलायो, अब्ला बिखौ बै ओनसोलनि आय खाजोना मावख 'गिरिजों बिजिरनाय जागोन जेराव बै जायगाया दं, एबा जेराव फालांगि, जिउ राहा एबा जिउ राहाया मोनसेनिफ्राय बांसिन जायगाफोराव सोलिबोगासिनो दं, बै ओनसोलनि आय खाजोना मावख' गिरिजों जेराव बिथांनि फालांगि, जिउ राहा एबा जिउ राहानि गुबै जायगाया दं। गुबुन गासैबो जाथायफोराव, खाजोना लाग्राखौ बिथाङा जाय ओनसोलाव थायो बेनि आय खाजोना मावख 'गिरिजों बिजिरनाय जागोन। गुबुन गुबुन आयदाफोरनि गेजेराव, दफा 64 नि दालाय दफा (5) जों बेखौ बिथोन होनाय जायो दि दालाय दफा (1) आरो (2) नि राहाफोरा जेराव जायखिजाया बिथोनजों होनाय एबा आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय आय
बिदी साप्लाइ कम्पानिनि केस (1) आव बे कर्टआ बिथोन होदोंमोन दि दफा 5 नि दालाय दफा 5 (7ए), जेरै बियो गुदि समाव दंमोन, मोनसे थि बोसोरनि थाखाय थाबथानाय केसखौ बारस्लायहोनो सानदोंमोन। बेखौ नोजोर होनाय जादोंमोन दि "बेबादि सोलायनाया मावखान्थिनि जायखिजाया थाखोयाव जानो हागौ", थारै थाबथानाय मावखान्थिखौ गनायो, आरो "थाखो" या बै मावखान्थिनि जागायजेन्नाय आरो जोबनायनि गेजेरनि मोनसे बिन्दोखौ फोरमायो। आरोबाव बे राहा दि बेबादि बारस्लायनाया गोनांथि जानाय नङा (1); 581 आय खाजोना मावख 'गिरि जायनिफ्राय केसखौ बारस्लायहोनाय जादोंमोन, बिनिफ्राय जारि खालामजानाय फोसावनाय फोसावथाइखौ फिन फोसावनाया रोखा महरै फोरमायो दि दालाय खोन्दोआ सान्ना लानाय बारस्लायनाया थारैनो मोनसे जायगानि आय खाजोना मावख' गिरिनि सिगाङाव थाबथानाय मोनसे थि केसखौ गुबुन जायगानि आय खाजोना मावख 'गिरिनाव बारस्लायहोनाय। "बिजिरसालिनि थिरांथाआ दफा 5 (ए) जों बाहायजानाय' आय खाजोना फोसाबनाय आइन 'नि दफा 5 (ए) नि गेजेरजों बाहायजानाय' सोदोबखौ सोलायदोंमोन।
बिथांनि आदालतआ मानिदोंमोन दि 'केस' सोदोबनि ओंथिया मोनसे थि बोसोरनि बिजिरसंनाय जाथाय। बे थिरांथा लानायनि उनाव आयेन आफादआ हेंथा होदोंमोन आरो 1956 मायथाइनि आय खाजोना फोसाबनाय आइन 26 नि गेजेरजों बियो मोनसे बुंफुरलु दाजाबदेरदोंमोन दि जायखिजाया सुबुंनि सोमोनदोआव 'केस' सोदोबनि ओंथिया जायखिजाया बोसोरनि सोमोनदोआव आय खाजोना आयेननि सिङाव गासिबो मावफारिफोर जाय सोलायसोल 'नि अक्ट' आव थाबथानाय जानो हागौ, आरो बे आयेननि सिङाव गासिबो मावफारिफोर दं जाय जायखिजाया बोसोरनि सोमोनदोआव सोलायसोल 'नि अक्ट' नि उनाव जागायनो हागौ। बिडी साप्लाइ कम्पानी (1) नि केसनि प्रिन्सिपलआ 1951 मायथाइनि बम्बे एक्ट 23 जों फोसाबजानाय कोड अफ क्रिमिनल प्रोसिजरनि दफा 14 नि सिङाव जारि खालामजानाय फोसावथायनि उदायथिखौ सान्ना नायनायाव जेबो सोमोनदो गैया। बिडी साप्लाय कम्पानिनि केस (1) आव बिजिरसंगिरिफोरा भारतारि आय आइननि राहाफोराव मोनसे आंखालनि मुलाम्फाखौ मोन्नो हादोंमोन, जेराव इयुननि खाजोना सोलायसोल 'नि थाखाय जेबो सरासनस्रा राहा गैया।
